About StormWatch
Data Sources
National Weather Service
api.weather.gov
Alerts, watches & warnings · 7-day forecast · Hourly forecast · Current observations · Radar station metadata · Point metadata (grid, zone, county)
Open-Meteo
api.open-meteo.com
Real-time temperature, humidity, dew point, wind, gusts, UV index, cloud cover, precipitation · Hourly forecast data for day modal charts · CAPE & CIN for tornado risk scoring · Precipitation totals (daily) · Gap-fills observation strip when NWS station data is unavailable
NOAA nowCOAST (NEXRAD MRMS)
nowcoast.noaa.gov/geoserver/observations/weather_radar
Real NEXRAD base reflectivity via NOAA WMS · Quality-controlled 1km MRMS mosaic · ~4-min updates · No API key · Works at all zoom levels · CONUS + Alaska, Hawaii, Caribbean, Guam
OpenWeatherMap Weather Maps 2.0
tile.openweathermap.org
Model-based map layers for Wind, Cloud cover, and Temperature · Hourly frames · Displayed as animated overlays on the radar map
NOAA Weather Radio
wxradio.org · weatherusa.net
Live audio streams of 86 NOAA All Hazards Radio stations · Location-matched to nearest transmitter · Volunteer-operated Icecast streams · Not for life-safety use
Nominatim / OpenStreetMap
nominatim.openstreetmap.org
ZIP code and city name geocoding to lat/lon coordinates
ip-api.com
ip-api.com
Approximate location fallback based on IP address when GPS is unavailable
USGS Water Services
waterservices.usgs.gov
Real-time stream gauge data for rivers and creeks near your location · Gage height readings every 15 min
NOAA National Hurricane Center
mapservices.weather.noaa.gov
Active tropical storm and hurricane tracks, forecast cone, wind radii, and movement data · Updated every 6 hours
Open-Meteo Air Quality
air-quality-api.open-meteo.com
US AQI with per-pollutant breakdown (PM2.5, PM10, Ozone, NO₂, CO) · Updated hourly
Tomorrow.io
api.tomorrow.io
Real-time weather code for accurate condition icons · Fires in parallel with NWS on every location load
Tornado Risk Model
The Risk tab scores 8 atmospheric factors using a combination of NWS forecast text (wind, hail, tornado mentions), real CAPE/CIN values from Open-Meteo, active alert counts, and time-of-day weighting. It is a supplemental awareness tool — always follow official NWS guidance.
Map
Radar map rendered with Leaflet.js using CartoDB dark basemap tiles and IEM NEXRAD radar overlay. Your location is shown as a yellow dot.
Privacy
No account required. No data is stored or transmitted beyond what's needed to fetch weather for your location. GPS coordinates are sent directly to NWS and Open-Meteo APIs — they are not logged by this app.
